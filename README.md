# Luna Case Website

## Description
Luna Case is a modern, responsive website for showcasing and selling phone cases. The project uses **HTML**, **Sass/SCSS**, **Bootstrap**, and **Python Flask** with Jinja2 templating.

The site includes:
- Home page with hero section
- Product catalog with grid layout
- Upload and wishlist sections
- User registration and login pages
- Admin panel for managing users and orders
- Shopping cart functionality

---

## Features
- Fully responsive design for desktop, tablet, and mobile
- Elegant and modern aesthetic using purple gradient themes
- SCSS organized into components: header, footer, home, products, services, upload, register, cart
- Interactive elements with hover effects and buttons
- Admin panel to monitor users and orders
- Login and registration forms
- Wishlist and upload functionality
